# 📂 ASP.NET MVC CV Projesi

Bu proje, **ASP.NET MVC 5** altyapısı kullanılarak geliştirilmiştir.  
Veri erişim katmanında **Generic Repository** ve **Design Pattern (Unit of Work, Dependency Injection)** gibi yaklaşımlar uygulanmıştır.  
Amaç; **katmanlı mimari**, **temiz kod** ve **yeniden kullanılabilirlik** prensiplerine uygun bir web uygulaması geliştirmektir.

---

## 🚀 Özellikler

- Kullanıcı dostu arayüz. ( Admin Paneli )
- Katmanlı mimari ile geliştirilmiş yapı  
- CRUD işlemleri (Create, Read, Update, Delete)
- Generic Repository kullanımı
- Unit of Work ile transaction yönetimi
- Dependency Injection ile bağımlılıkların yönetimi  
- SQL Server ile veritabanı işlemleri 
- Responsive tasarım (Bootstrap)
